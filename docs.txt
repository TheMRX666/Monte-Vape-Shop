shop.sql - файл для бази даних MySQL
server - серверна частина сайту
client - клієнт частина сайту

Запуск:
папка server: php -S localhost:4200
після того як сервер запустився, відкриваємо нову консоль
папка client:
npm i, очікуємо скачування всіх залежностей, потім npm run start

База даних має бути запущена через OpenServer
